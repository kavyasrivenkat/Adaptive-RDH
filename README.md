# Adaptive-RDH

**A Final Year Project**

## Overview

Adaptive-RDH is a Python-based project developed as a final year academic project. This repository contains the implementation and documentation for the adaptive RDH (Reversible Data Hiding) system.

## Project Description

This project focuses on implementing an adaptive approach to Reversible Data Hiding (RDH), which is a technique used to embed data into digital media (typically images) while maintaining the ability to perfectly recover the original media and extract the hidden data.
